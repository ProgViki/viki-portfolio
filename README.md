# Personal Portfolio Website

A modern, enterprise-grade personal portfolio website built with Next.js, TypeScript, and Tailwind CSS. Features a stunning UI with VS Code Blue and Emerald Green theme, interactive components, and a focus on showcasing enterprise architecture expertise.

## 🚀 Tech Stack

### Core Technologies
| Technology | Purpose |
|------------|---------|
| **Next.js 14** | React framework with App Router |
| **TypeScript** | Type-safe JavaScript |
| **Tailwind CSS** | Utility-first CSS framework |
| **Framer Motion** | Smooth animations and transitions |
| **React Hook Form** | Form management with validation |
| **Zod** | Schema validation for forms |

### UI & Styling
| Library | Purpose |
|---------|---------|
| **Lucide React** | Beautiful, consistent icons |
| **React Fast Marquee** | Scrolling tech stack animation |
| **React Icons** | Additional icon set |
| **Custom Gradients** | VS Code Blue (#007ACC) and Emerald Green (#10B981) |

### Deployment
| Platform | Purpose |
|----------|---------|
| **Vercel** | Optimized for Next.js hosting |
| **Netlify** | Alternative hosting option |
